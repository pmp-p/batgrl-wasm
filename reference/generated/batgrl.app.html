If provided, stderr is written to this path.

render_modeLiteral[“regions”, “painter”], default: “regions”

Determines how the gadget tree is rendered. "painter" fully paints every gadget back-to-front. "regions" only paints the visible portion of each gadget. "painter" may be more efficient for a large number of non-overlapping gadgets.
