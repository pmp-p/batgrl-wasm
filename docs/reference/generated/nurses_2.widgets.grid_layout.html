Raises
ValueError

If grid is full and add_widget() is called.

Notes

Re-ordering children (such as through pull_to_front()) and calling _reposition_children() will change the positions of the children in the grid.

The read-only attribute minimum_grid_size is the minimum size the grid must be to show all children. This can be used to set the size of the grid layout, e.g., my_grid.size = my_grid.minimum_grid_size.

Widget properties’ target values. E.g., to smoothly tween a widget’s position to (5, 10) over 2.5 seconds, specify the pos property as a keyword-argument: await widget.tween(pos=(5, 10), duration=2.5, easing=Easing.OUT_BOUNCE)

Warning

Running several tweens on the same properties concurrently will probably result in unexpected behavior. tween won’t work for ndarray types. If tweening size or pos hints, make sure the relevant hints aren’t None to start.

class nurses_2.widgets.grid_layout.Orientation(value)#

Bases: str, enum.Enum

Orientation of the grid.

As an example, the orientation LR_TB means left-to-right, then top-to-bottom. Orientation is one of “lr-tb”, “lr-bt”, “rl-tb”, “rl-bt”, “tb-lr”, “tb-rl”, “bt-lr”, “bt-rl”.
